Detailed Description

Robot interface functions for usage in the server.c file.

Use these functions to operate the robot. The robot can either be a 'real' robot (VolksBot Indoor), or a simulated BubbleRob.

Created on: 12.03.2014 Author: fengelha, lindhors

Read a single image from the robot's camera.

Parameters
bufPointer to a char*, which will return the pointer to a buffer containing the image data as windows bmp image. You may write this data immediately to a file. The pointer to the buffer is valid until the next call to getRobCamImage().
sizePointer to an integer, where the actual size in bytes of the image is returned.

Send an alive-signal to the robot.
